# Founder's Map

## Install

* Install dependencies

  ```sh
  $> npm install -g gulp bower babel # optional, if you'd like to have CLI tools in your env
  $> npm install
  $> bower install
  ```

## Run Tests

* Run Tests

  ```sh
  $> gulp test
  ```

* Runt Tests With Watch

  ```sh
  $> gulp test:watch
  ```

## Development

* Develop with live reload in browser

  ```
  $> gulp serve
  ```
